Using custom CSS with bootstrap

I use bootstrap and css together. In order for them to work together, I create a file called 'custom.css' and place it in my '/static' directory. The basic template I use for my nav bar can be found here. I use chrome, and in order to see my page update in development mode when I change my custom css I need to delete my chrome cache. To do this, press 'ctr'-'shift'-'del' and delete cache.
