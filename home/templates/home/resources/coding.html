Git
Useful Commands
git add .
Adds all changes from last commit to stage.
git commit -m "description of changes"
Commits all changes on stage to branch. Must write in some description and must use double quotes.
git status
Displays branch name and staging.
git init
initialized git in current directory
git pull
merges remote repository into current directory
git remote
lists remote repositories
git checkout 'branch_name'
switches to that branch
